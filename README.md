## 概要

omemoji のポートフォリオ

## 使用技術

- Next.js
- Tailwind CSS
- TypeScript
- Cloudflare Pages

## 実装する機能

### 全般

- デザインをモダンで統一感あるものにする

### About

- 自分のプロフィールやアカウント、実績などを載せる

### Artwork

- 個別の作品ページ
- 作品一覧
- Tag 機能

### Articles

- Markdown で記事を書けるようにする
- Tag 機能
- リンクカード
- Zenn 風

### これから実装していきたい機能

- 横スクロールの作品選択欄（Pixiv みたいな）
